Get the app
To install on Android:
1. Tap the green button (downloads the app).
2. If it warns about “unknown apps,” tap Settings → Allow, then Install.
3. Open it, sign up, pick Handball. 🖐️
To install on iPhone:
1. Tap the green button — it opens TestFlight (free Apple app, installs if needed).
2. Tap Accept, then Install.
3. Open it, sign up, pick Handball. 🖐️
